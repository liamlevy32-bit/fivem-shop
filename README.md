# FiveM Premium Shop

חנות דיגיטלית מתקדמת לשרתי **FiveM**, המיועדת למכירה של VIP, חבילות התחלה, כסף, רכבים ותוכן ייחודי בתוך השרת.

הפרויקט משלב עיצוב מודרני בסגנון גיימינג עם תשתית טכנולוגית מתקדמת, ומיועד לספק חוויית קנייה חלקה, מהירה ואמינה לשחקנים ולמנהלי שרתים.

---

## 🚀 תכונות עיקריות

* 🛒 חנות דיגיטלית למוצרי FiveM
* 💳 תשלומים מאובטחים (PayPal – Sandbox / Live)
* 🤖 חיבור אוטומטי ל-Discord
* 🎮 מוכן לחיבור לשרת FiveM
* 🎨 עיצוב גיימינג מודרני ורספונסיבי

---

## 🧰 טכנולוגיות

* HTML / CSS / JavaScript
* Node.js (Backend)
* PayPal API
* Discord.js

---

## 📂 מבנה הפרויקט

```
fivem-shop/
│─ server.js
│─ package.json
│─ .env
│─ public/
│   └─ index.html
```

---

## ▶️ הרצה מקומית

```bash
npm install
node server.js
```

לאחר מכן פתח בדפדפן:

```
http://localhost:3000
```
